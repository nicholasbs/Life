#About

Conway's Game of Life in Haskell. We wrote this to learn Haskell so it's
probably pretty bad.

Thanks to Bartlomiej Niemtur whose code we used when figuring out how to
display the board (https://github.com/nbartlomiej/gameofhaskell)
